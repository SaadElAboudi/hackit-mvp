# Hackit MVP - Flutter Frontend

Ce dossier contient une migration Flutter du frontend existant.

Prerequis:
- Flutter SDK installé (stable channel)

Lancer l'app en local (dans ce dossier):

```bash
flutter pub get
flutter run
```

API backend par défaut: http://localhost:3000 (peut être changé dans `lib/services/api_service.dart`).
